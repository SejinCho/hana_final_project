1. 깃 허브에서 새 저장소를 만듭니다. (https://github.com/)

1, Create a new repository on Github. (https://github.com/)



2. Git Bash를 엽니다.

2. Open Git Bash.



3. cd to/your/project/directory 현재 작업하고 있는 디렉토리로 이동합니다.

3. Go to the current working directory to your local project.



4. 해당 디렉토리를 깃레포로 초기화해줍니다.

4. Initialize the local directory as a Git repository.

$ git init

5. 로컬에 있는 파일들을 추가하고 커밋합니다. 

5. Add the files in your new local repository and Commit the files.

$ git add .
$ git commit -m "First Commit"

6. 로컬 프로젝트 파일에 원격 저장소의 주소를 설정해줍니다. 

6. Set URL of the remote repository.

$ git remote add origin [ github Clone URL ]

7. 로컬 레포에서 있었던 변경사항들을 리모트 레포로 push합니다.

7. Push the changes in your local repository to Github.

$ git push origin master
